## How to Upgrade

Follow these steps to update your Annelise Lawson O'BAILEY instance.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run the version check command in your project root.
````bash
annelise version
````
  </Step>
  <Step title="Update Package" icon="download">
    Use your preferred package manager.
    <CodeGroup tabs="npm,yarn">
````bash
npm install annelise@latest
````
````bash
yarn add annelise@latest
````
    </CodeGroup>
  </Step>
  <Step title="Restart Server" icon="refresh-cw">
    Restart your documentation server to apply changes.
````bash
npm run dev
````
  </Step>
  <Step title="Review Changes" icon="book-open">
    Read the full changelog above and test your pages.
  </Step>
</Steps>

<Callout kind="tip">
  After upgrading, validate your MDX pages for parsing errors, especially around special characters like `{API_KEY}` or `<100ms`.
</Callout>
