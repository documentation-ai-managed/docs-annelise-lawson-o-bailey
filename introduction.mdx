## Overview

Annelise Lawson O'BAILEY provides a centralized hub for your project documentation. You create, organize, and collaborate on docs effortlessly. This platform supports markdown editing, version control, and search across all your content. Start by uploading files or building pages from scratch to streamline your workflow.

<Callout kind="info">
  Use the brand color `<span style="color: #3B82F6;">#3B82F6</span>` (Annelise Blue) for headers, buttons, and accents to maintain visual consistency.
</Callout>

## Quick Start

Get up and running in minutes. Follow these steps to create your first document.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://annelise-lawson-o-bailey.example.com` and create an account with your email.
  </Step>
  <Step title="Create a Space" icon="plus">
    Click the new space button and name it, like "Project Alpha Docs".
  </Step>
  <Step title="Add Your First Page" icon="file-plus">
    Inside the space, create a new MDX page. Use the editor to add content.
  </Step>
  <Step title="Publish and Share" icon="share-2">
    Save and publish. Share the public link with your team.
  </Step>
</Steps>

<CodeGroup tabs="CLI,API">
  ```bash
  # Install CLI tool
  npm install -g annelise-cli

  # Login and create space
  annelise login
  annelise space create "Project Alpha"
  ```
  ```javascript
  // API example to create a space
  const response = await fetch('https://api.example.com/v1/spaces', {
    method: 'POST',
    headers: { 'Authorization': `Bearer ${YOUR_API_KEY}` },
    body: JSON.stringify({ name: 'Project Alpha' })
  });
  ```
</CodeGroup>

## Brand Guidelines

Follow these guidelines to keep your docs on-brand:

- **Primary Color:** `#3B82F6` for links, buttons, and highlights.
- **Typography:** Use sans-serif fonts like Inter or system defaults.
- **Icons:** Stick to Lucide icons for consistency.

<Callout kind="tip">
  Test color contrast with tools like WebAIM to ensure accessibility. Always preview pages before publishing.
</Callout>
